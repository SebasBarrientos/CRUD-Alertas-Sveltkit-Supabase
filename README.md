# Prueba técnica   
### Proyecto utilizando Supabase, Sveltekit y Tailwind (junto con DaisyUi) para la creacion de una interfaz para ver, cargar y modificar alertas. Tambien se agrego un hipervinculo a Apache Guacamole.
